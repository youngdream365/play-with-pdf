XML Print and Page Layout Working Group

Public Page

Publications • Background• Participation and Discussion• IPR DIsclosures• Implementations

This is the public web page for the XML Print and Page Layout Working Group XPPL) Working Group of the World Wide Web Consortium (W3C). The group is part of W3C's XML Activity and does its technical work in Member-only space.

The XPPL WG is really a continuation of the XSL-FO work that was formerly part of the XSL Working Group.

This Working Group is no longer active, because of insufficient participation. The specifications are no longer maintained.

The group's charter covers the period until the end of January 2013.

Publications

XSL-FO 1.1 was published in 2006 by the XSL Working Group.

Requirements for Japanese Text Layout was published by a task force in 2009.

There is an XSL-FO 2.0 Draft.
